For both games, you need the lowest settings to run them. They are laggy at first, but it gets better. Set the Mipmap level to zero and set particles to Minimal. Also, TPS (Ticks Per Second) is more important than FPS (Frames Per Second). TPS affects how the game runs, so keep it high. Additionally, water can cause lag in the game
